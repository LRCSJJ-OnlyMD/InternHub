Step 1: Register a New Account

What happens:

Creates a new user account and sends a verification email.

Step 2: Verify Email

What happens:

Activates your account using the token from the email.

Step 3: Login

What happens:

Logs you in and receives a JWT token for authenticated requests.

Step 4: Enable Two-Factor Authentication (2FA)

What happens:

1. Generates a QR code for Google Authenticator
2. Open Google Authenticator app on your phone
3. Tap "+" or "Scan QR code"
4. Point your camera at the QR code below
5. The app will add "AuthApp (your-email)" and show a 6-digit code

Scan this QR Code with Google Authenticator:

After scanning, your authenticator will show a 6-digit code that changes every 30 seconds.

Step 5: Test Login with 2FA

What happens:

After enabling 2FA, you'll need to provide the 6-digit code from Google Authenticator when logging in.

1. Logout first
2. Go back to Step 3
3. Enter your email and password
4. You'll see a 2FA code field
5. Open Google Authenticator and enter the current 6-digit code
